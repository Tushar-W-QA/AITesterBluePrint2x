Test Plan: VWO Login Dashboard

Anti-Hallucination Verification

  • Verified Features: Email, MFA, SSO, Passkey, Registration.
  • Security: HTTPS, Rate limiting, GDPR.
  • Performance: < 2s load time.

1. Objective

Ensure secure and accessible authentication entry for VWO platform users across all supported environments.

2. Scope

Testing Types

  • Manual Functional & Usability
  • Automated UI (Playwright) & API (REST)
  • Security Audit (OWASP)
  • Performance Benchmarking

3. Environments

  • Browsers: Chrome, Safari, Edge, Firefox.
  • OS: Windows, macOS, iOS, Android.

4. Defect Management

All defects to be logged in JIRA with severity, steps to reproduce, and visual evidence. High-priority bugs must be resolved before release sign-off.

5. Entry/Exit Criteria

Entry: Stable build and approved test cases.

Exit: 100% execution, 0 Critical/High open defects.
